## 各題重點

- **題1**：九九乘法表按特定方向讀取，數字統計與連乘積計算
- **題2**：`random.seed()` 控制亂數、list 去重、大整數冪次運算
- **題3**：`random.random()` 生成發票號碼、由後往前對號規則
- **題4**：`datetime.date` 星期幾判斷、民國年轉西元年
- **題5**：正規表達式分類四種數字格式（純數字 / 貨幣逗號格式）
- **題6**：`collections.Counter` 統計重複姓名、姓氏篩選
- **題7**：成績不及格科目計數、總分頻率統計
- **題8**：大整數相乘後對數字字串做非重疊子字串搜尋

---

## 作答注意事項

1. **題1 乘法表讀取方向**：須從第 9 行倒序往第 1 行讀，且每格包含被乘數、乘數、結果三個數字。
2. **題5 型態2加總**：分類正規表達式須精確，避免重複計算造成多兩位數。
3. **題8 a×b**：`a` 與 `b` 是**整數相乘**（不是字串拼接），需先 `int()` 再 `*`，最後對乘積字串做分析。
